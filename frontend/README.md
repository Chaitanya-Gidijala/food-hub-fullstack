# FoodHub Frontend - Angular Application

## Prerequisites
- Node.js 18+
- Angular CLI 17

## Installation
```bash
npm install
```

## Running the Application
```bash
npm start
```
Application will run on http://localhost:4200

## Features
- User Authentication (Login/Register)
- Browse Restaurants
- View Menu Items
- Add to Cart
- Stripe Payment Integration
- Order Management
- Order History

## Material Design
- Angular Material components
- Material Icons
- Responsive design

## API Endpoints
- User Service: http://localhost:8081
- Restaurant Service: http://localhost:8084
- Order/Cart Service: http://localhost:8083
- Payment Service: http://localhost:8082

## Build for Production
```bash
npm run build
```
